🧠 RehabEase – AI-Based Physiotherapy Assistant
📌 Overview

RehabEase is an AI-powered rehabilitation assistant designed to help users perform physiotherapy exercises correctly using real-time pose detection. It leverages computer vision and machine learning to analyze body posture and provide feedback, making recovery more efficient and accessible.

This system can be used in physiotherapy, fitness training, and injury recovery where maintaining correct posture is critical.

🚀 Features
🎥 Real-time Pose Detection using MediaPipe
🧍‍♂️ Exercise Monitoring via webcam/video input
📊 Posture Analysis & Feedback
🔗 Full Stack Integration (Frontend + Backend + ML)
⚡ REST API Support for ML model interaction
🌐 Scalable for healthcare & fitness applications

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/HharshMore/Rehabease.git
cd RehabEase-main
2️⃣ Install Backend Dependencies
npm install
3️⃣ Setup Python Environment
pip install -r requirements.txt
4️⃣ Run Flask ML Server
python app.py
5️⃣ Run Node.js Server
npm start
6️⃣ Open in Browser
http://localhost:3000
